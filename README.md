# Python Mini Project – Job Description Analyzer

This project analyzes multiple job descriptions using Python to extract important skills and role requirements through basic text processing.

## Objective
To read and process job description text files and identify key skills commonly required across different roles.

## Tools & Concepts Used
- Python
- File Handling
- Text Processing
- Jupyter Notebook

## Project Structure
- `working_file.ipynb` – main analysis notebook  
- `files/` – job description text files  
- `main.py` – Python script version  

## How to Run
1. Open `working_file.ipynb` in Jupyter Notebook
2. Run the cells sequentially
3. View extracted insights from job description files
